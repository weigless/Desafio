2) Dado a sequência de Fibonacci:

Escreva um programa na linguagem que desejar onde, informado um número, ele calcule a sequência de Fibonacci e retorne uma mensagem avisando se o número informado pertence ou não a sequência.

Resposta:

def fibonacci_sequence(n):
    n1, n2 = 0, 1

    if n == n1 or n == n2:
        return f"{n} pertence à sequência de Fibonacci."

    while True:
        nth = n1 + n2
        if nth == n:
            return f"{n} pertence à sequência de Fibonacci."
        elif nth > n:
            return f"{n} não pertence à sequência de Fibonacci."
        n1, n2 = n2, nth

try:
    numero = int(input("Informe um número inteiro: "))
    resultado = fibonacci_sequence(numero)
    print(resultado)
except ValueError:
    print("Por favor, insira um número inteiro válido.")

4) Você está em uma sala com três interruptores:

Você não pode ver as lâmpadas da sala em que está, mas pode ligar e desligar os interruptores quantas vezes quiser. Seu objetivo é descobrir qual interruptor controla qual lâmpada. Como você faria para descobrir, usando apenas duas idas até uma das salas das lâmpadas, qual interruptor controla cada lâmpada?

Resposta: Para resolver esse problema, eu faria o seguinte: Na primeira ida, ligaria o interruptor A e deixaria ligado por alguns minutos. Em seguida, desligaria o interruptor A e ligaria o interruptor B. Na segunda ida, observaria as lâmpadas na sala: Se uma lâmpada estiver acesa, saberei que o interruptor A controla essa lâmpada. Se uma lâmpada estiver apagada, mas estiver quente, isso significa que o interruptor B controla essa lâmpada. E, se a última lâmpada estiver apagada e fria, concluirei que o interruptor C controla essa lâmpada. Dessa forma, em apenas duas idas à sala das lâmpadas, posso determinar qual interruptor controla cada uma delas.

5) Escreva um programa que inverta os caracteres de um string.

def inverter_string(string):
    string_invertida = ""
    
    for i in range(len(string) - 1, -1, -1):
        string_invertida += string[i]
    
    return string_invertida

texto_original = input("Digite uma string para inverter: ")
texto_invertido = inverter_string(texto_original)
print("String invertida:", texto_invertido)
